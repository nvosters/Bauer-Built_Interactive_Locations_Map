GitHub Pages setup

Files in this folder:
- index.html
- Bauer Built Interactive Locations Map.csv

How to publish:
1. Create a GitHub repository.
2. Upload BOTH files to the root of the repository.
3. In GitHub, go to Settings > Pages.
4. Under Build and deployment:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Save.
6. Wait about 1-2 minutes.
7. Your map will be live at:
   https://YOUR-USERNAME.github.io/REPOSITORY-NAME/

How to update later:
1. Replace 'Bauer Built Interactive Locations Map.csv' in the repo with the updated file.
2. Commit the change.
3. Refresh the website after GitHub Pages republishes.

Important:
- Keep the CSV filename exactly:
  Bauer Built Interactive Locations Map.csv
- Keep index.html and the CSV in the same folder.
